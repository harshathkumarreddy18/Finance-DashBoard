## 📖 Project Overview

The **Personal Finance Dashboard** is an interactive Business Intelligence solution built using **Microsoft Power BI** to help users gain meaningful insights into their financial activities.

This dashboard consolidates financial data into easy-to-understand visualizations, allowing users to monitor income, expenses, savings, and transaction history in real time. By combining data modeling, DAX calculations, and interactive reporting, the dashboard enables users to make informed financial decisions based on clear and actionable insights.

---

## 🎯 Objectives

- Track total income, expenses, and savings.
- Monitor monthly financial performance.
- Identify major spending categories.
- Analyze transaction history.
- Support better budgeting and financial planning.
- Deliver an interactive and user-friendly reporting experience.

---

# 📊 Dashboard Pages

## 🏠 Finance Overview

The Finance Overview page provides a high-level summary of overall financial performance through interactive KPIs and visual reports.

### Features

- 💵 Total Income
- 💸 Total Expenses
- 💰 Net Savings
- 📈 Monthly Income Trends
- 📉 Monthly Expense Trends
- 🥧 Expense Distribution by Category
- 📅 Time-Based Analysis
- 🎛 Interactive Slicers and Filters

---

## 💳 Transaction Analysis

The Transaction Analysis page provides a detailed breakdown of every financial transaction.

### Features

- Complete transaction history
- Income and expense records
- Category-wise analysis
- Date filtering
- Search functionality
- Interactive transaction table
